General Social Media

General #

0.0 # General

  1. 0.0.1 # The purpose of this Policy is to provide standards for the exercise of the Chief Returning Officer (CRO)'s authority under Bylaw 3 as well as their executive authority.

0.1 # Deputy Returning Officers

  1. 0.1.1 # The CRO shall, to the extent possible, only assign DROs to candidates with which they have no personal connection. DRO candidates shall disclose such election candidates to the CRO during the selection process.
  2. 0.1.2 # DROs are obligated to surveil their assigned candidates' campaign activity and report breaches to the CRO.

0.2 # Excusing of Breaches

  1. 0.2.1 # As a general principle, the CRO may only excuse breaches which do not cause harm to the integrity of the election.
  2. 0.2.2 # Breaches of the following sections of Bylaw 3 always cause harm to the integrity of the election and shall never be excused:
    1. 0.2.2.a # Section 8.0.1
    2. 0.2.2.b # All of Section 8.0.3
    3. 0.2.2.c # Section 8.1.1
    4. 0.2.2.d # Section 8.1.2
    5. 0.2.2.e # Section 8.1.3
    6. 0.2.2.f # Section 8.1.5.a
    7. 0.2.2.g # Section 8.1.5.d
    8. 0.2.2.h # Section 8.1.7
    9. 0.2.2.i # All of Section 8.2.1
    10. 0.2.2.j # All of Section 8.2.2
    11. 0.2.2.k # Section 8.2.5
    12. 0.2.2.l # Section 8.3.1
    13. 0.2.2.m # All of Section 8.3.2
    14. 0.2.2.n # Section 8.3.5
    15. 0.2.2.o # Section 8.4.2
    16. 0.2.2.p # Section 8.4.4
    17. 0.2.2.q # Section 8.6.4.a
    18. 0.2.2.r # Section 8.6.5
    19. 0.2.2.s # All of Section 8.7
  3. 0.2.3 # In considering whether to excuse the breaches listed in section 0.2.4, the CRO shall account for all the following factors (in no particular order):
    1. 0.2.3.a # The extent of the impact of the breach, such as how long the breach was visible, or how many voters saw the breach
    2. 0.2.3.b # The extent to which the candidate had control over any other involved party (in the CRO's sole judgement)
    3. 0.2.3.c # The mens rea of the candidate
  4. 0.2.4 # The breaches to which section 0.2.3 applies include:
    1. 0.2.4.a # Bylaw 3, section 8.3.4
    2. 0.2.4.b # Social media campaigning rules not listed above

Social Media #

1.0 # General

  1. 1.0.1 # These social media rules are to be presented during each All Candidates Meeting and provide clarity to Bylaw 3 section 8.2 and section 8.6.
  2. 1.0.2 # Friend/follow/connection/etc. requests are considered campaigning.

1.1 # Instagram

  1. 1.1.1 # Instagram campaign accounts must be newly created, and may initially be private. They must become public by the time the Campaign Period begins.
  2. 1.1.2 # Instagram campaign accounts shall be set up so that sharing/reposting of stories and posts is disabled.
  3. 1.1.3 # Reels are considered videos and are prohibited under Bylaw 3 section 8.6.3.

1.2 # Discord

  1. 1.2.1 # The following rules apply to Discord campaign servers created by candidates:
    1. 1.2.1.a # The server must be created and owned by a Discord campaign account.
    2. 1.2.1.b # The ORO must approve the server before anyone else may be invited.
    3. 1.2.1.c # Candidates may only use approved Discord campaign accounts within the server.
    4. 1.2.1.d # All content in the server posted by candidates is considered campaign material.
    5. 1.2.1.e # As long as "vote.skule.ca" is included in the description of every channel, it is considered included in any other campaign material in the server.
    6. 1.2.1.f # Voters may only be invited to the server using approved social media campaigning accounts, and invitations to the server are considered campaign material.
  2. 1.2.2 # The following rules apply to Discord servers created by non-candidates (including the ORO):
    1. 1.2.2.a # The server must be created for the purpose of platforming all candidates for one or more elections.
    2. 1.2.2.b # The rules of the server must be approved by the ORO.
    3. 1.2.2.c # Candidates must follow the rules of the server.
    4. 1.2.2.d # Candidates may only campaign for a given election in the server if all candidates for that election are present in the server.
    5. 1.2.2.e # Each election to which the server applies must be logically separate from the others (e.g. in a different channel category).
    6. 1.2.2.f # The text "vote.skule.ca" must be included in the description of every channel, and is thus considered to be included in other campaign material in the server.
    7. 1.2.2.g # Candidates are not required to use a campaign account to join the server and campaign within it.
    8. 1.2.2.h # Voters may be invited to the server by any means, including in other Discord servers (in accordance with the rules of those servers) and with personal social media accounts, as long as no reference is made to any candidates.
  3. 1.2.3 # Campaigning is not permitted in any other Discord servers.
  4. 1.2.4 # Discord campaign accounts must be newly created and shall not send messages in any server other than those they create under section 1.2.1
  5. 1.2.5 # Animated images (including GIFs, APNGs, and WebMs), animated stickers, and animated emojis are considered videos and are prohibited under Bylaw 3 section 8.6.3.
